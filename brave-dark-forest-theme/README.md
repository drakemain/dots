# Dark Forest Theme for Brave

A custom Brave browser theme matching your Dark Forest nvim/neovide, ghostty, and zsh configuration.

## Color Palette

This theme uses the same Dark Forest color scheme:
- **Background**: `#1a1f1a` - Deep forest green
- **Foreground**: `#d4d9c7` - Light muted cream
- **Accents**: Natural earthy tones (forest greens, muted browns, slate blues)

## Installation

1. Open Brave and navigate to `brave://extensions/`
2. Enable "Developer mode" using the toggle in the top-right corner
3. Click "Load unpacked"
4. Select the `brave-dark-forest-theme` directory
5. The theme will be applied immediately

## Uninstallation

1. Go to `brave://extensions/`
2. Find "Dark Forest" in the extensions list
3. Click "Remove"

## Customization

To modify colors, edit the `manifest.json` file:
- Colors are specified as RGB arrays: `[R, G, B]`
- Changes take effect after clicking the reload button in `brave://extensions/`
